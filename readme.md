# Prerequisites

 - Node.js
 - NPM

# Installing

`npm install`

# Building

 - `npm run start` for building and watching, then open your browser at http://localhost:8080
 - `npm run build` to build only

# Testing

 - `npm run test:watch`
 - few seconds after, in another terminal tab `npm run test` to launch protractor
