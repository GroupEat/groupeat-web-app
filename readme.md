# Prerequisites

 - Node.js
 - NPM
 - Gulp

# Installing

`npm install`

# Updating

`gulp pull`

# Building

 - `gulp` for building and watching (livereload)
 - `gulp build` to build only

 # Testing

 `gulp test`
